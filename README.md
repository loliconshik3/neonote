# Description

CLI note manager, with some interesting features.

# Features

* Interface on pyfzf.

* Write & rewrite notes with any length.

* Use any editor for write your notes.

# Install

1. Install requirments: `pip install -r requirments.txt`

2. Start sh file: `sh start.sh`
